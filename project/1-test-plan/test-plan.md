# Test Plan

## Objective
Ensure quality of Login, Cart, and Checkout features.

## Scope
- Functional Testing
- API Testing
- Automation Testing

## Out of Scope
- Performance testing
- Security deep testing

## Test Strategy
- Manual test cases
- API validation via Postman
- Automation via Cypress

## Entry Criteria
- Application deployed
- Requirements defined

## Exit Criteria
- All critical bugs fixed
- 90% test cases passed

## Risks
- Unstable environment
- Incomplete requirements

## Tools
- Postman
- Cypress
- GitHub
